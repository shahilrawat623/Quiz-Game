# 🎮 Quiz Game

An interactive quiz application built using **HTML, CSS, and JavaScript**.  
Players can answer multiple-choice questions, track their score, and enjoy a smooth, responsive interface.

---

## 📌 Features
- Multiple-choice questions with instant feedback
- Score tracking system
- Responsive design (works on desktop & mobile)
- Clean UI with engaging animations
- Easy to extend with new questions

---

## 🛠️ Technologies Used
- **HTML5** – Structure
- **CSS3** – Styling & responsiveness
- **JavaScript (ES6)** – Game logic
